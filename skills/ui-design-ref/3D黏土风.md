【整体风格】
3D 黏土质感 + 柔和粉彩 + 圆润膨胀感

【适用应用】
儿童教育、健康生活、社交娱乐、轻量工具、趣味电商

【配色】
主色：#7C5CFC（黏土紫）
辅色：#FF8FAB（黏土粉）
点缀：#FFCB57（黏土黄）
成功色：#5ED6A0（黏土绿）
背景：#F0EDFA（浅薰衣草灰）
卡片：#FFFFFF
文字主：#2D2B3D（深紫灰）
文字副：#8B87A0（中灰紫）
文字白：#FFFFFF

【布局】
页面容器：flex: 1, backgroundColor: '#F0EDFA'
内容滚动：`<ScrollView>`
  contentContainerStyle: paddingBottom: 100
栅格间距：marginBottom: 20
卡片区域：paddingHorizontal: 20
响应式：useWindowDimensions()，大屏加大留白

【核心样式手法】
黏土凸起阴影（核心）：
  外层阴影模拟 3D 凸出——
  iOS:
    shadowColor: '#7C5CFC',
    shadowOffset: { width: 0, height: 8 },
    shadowOpacity: 0.25,
    shadowRadius: 16
  Android:
    elevation: 8
  内描边模拟高光：
    borderWidth: 1.5,
    borderColor: 'rgba(255,255,255,0.6)'
  → 白色半透内边 + 彩色外阴影 = 黏土膨胀感

大圆角：
  卡片 borderRadius: 24
  按钮 borderRadius: 20
  头像/图标容器 borderRadius: 9999（全圆）
  → 一切棱角磨圆，模拟手捏黏土

柔色渐变背景块：
  `<LinearGradient`
    colors={['#F0EDFA','#E8E3F8','#F0EDFA']}
    start={{x:0,y:0}} end={{x:0,y:1}}>
  → 模拟柔光照射在黏土表面

图标气泡底座：
  width: 56, height: 56, borderRadius: 20,
  backgroundColor: '#EDE8FF',（或各色淡底）
  alignItems: 'center', justifyContent: 'center',
  borderWidth: 1.5, borderColor: 'rgba(255,255,255,0.7)',
  shadowColor: '#7C5CFC', shadowOffset:{width:0,height:4},
  shadowOpacity: 0.18, shadowRadius: 8

文字排版：
  标题 fontSize: 22, fontWeight: '800', color: '#2D2B3D'
  正文 fontSize: 15, fontWeight: '500', color: '#8B87A0', lineHeight: 22
  → 粗圆字重，配合圆润视觉

【技术栈】
图标：@expo/vector-icons（MaterialCommunityIcons / Ionicons — 圆润填充风格）
渐变：expo-linear-gradient（柔色背景、按钮渐变）
字体：expo-font 加载 Nunito（圆润无衬线全系列）
动画：react-native-reanimated（弹性入场：
  withSpring({ damping: 12, stiffness: 120 })，
  入场 scale: 0.85→1 + translateY: 20→0，
  按压 scale: withSpring(0.95) 模拟按捏手感）
样式方案：StyleSheet.create
