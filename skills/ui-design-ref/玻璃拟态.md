【整体风格】
毛玻璃透明 + 光影折射 + 轻盈通透

【适用应用】
天气/仪表盘、音乐播放器、社交资料页、金融概览、智能家居

【配色】
主色：#6C63FF（极光紫）
辅色：#00D2FF（冰蓝）
点缀：#FF6B9D（霓虹粉）
背景渐变：['#0F0C29','#302B63','#24243E']（深空渐变底）
卡片填充：rgba(255,255,255,0.08)
卡片边框：rgba(255,255,255,0.15)
文字主：#FFFFFF
文字副：rgba(255,255,255,0.6)

【布局】
页面容器：flex: 1，底层铺 <LinearGradient> 全屏渐变
内容滚动：<ScrollView> contentContainerStyle: paddingBottom: 100
栅格间距：marginBottom: 20
正文区域：paddingHorizontal: 20
响应式：useWindowDimensions()，大屏加大留白

【核心样式手法】
毛玻璃卡片（核心）：
backgroundColor: 'rgba(255,255,255,0.08)',
borderWidth: 1,
borderColor: 'rgba(255,255,255,0.15)',
borderRadius: 24,
overflow: 'hidden'
→ RN 无原生 blur，用半透明底 + 亮边模拟玻璃层

高光内描边：
borderWidth: 1,
borderColor: 'rgba(255,255,255,0.18)'
→ 上/左偏亮，模拟光照折射

光斑装饰（背景点缀）：
position: 'absolute', width: 200, height: 200,
borderRadius: 100,
backgroundColor: '#6C63FF',（或 '#00D2FF'/'#FF6B9D'）
opacity: 0.3
→ 多个彩色圆形散布在背景层，制造光斑朦胧感

渐变描边按钮：
外层 <LinearGradient> borderRadius: 20, padding: 1.5
内层 View backgroundColor: 'rgba(15,12,41,0.8)', borderRadius: 18.5
→ 模拟渐变边框

文字排版：
标题 fontSize: 26, fontWeight: '700', color: '#FFF'
正文 fontSize: 14, fontWeight: '400', color: 'rgba(255,255,255,0.6)', lineHeight: 21

【技术栈】
图标：@expo/vector-icons（Ionicons — 线性风格）
渐变：expo-linear-gradient（背景、光斑、渐变描边）
模糊：expo-blur BlurView（iOS 真毛玻璃；Android 回退半透明）
字体：expo-font 加载 Inter（现代无衬线）
动画：react-native-reanimated（
入场 opacity: 0→1 + scale: 0.95→1, duration: 600,
Easing.out(Easing.cubic)）
样式方案：StyleSheet.create
