新拟态双层阴影 + 暖灰白统一底色 + 大圆角胶囊感 + 渐变按钮

【适用应用】
AI 工具、生活服务、SaaS、在线教育、习惯养成、健康管理

【核心样式手法】
★ 凸起卡片必须两层嵌套 View 实现双阴影 ★
```
<View style={styles.shadowDark}>
  <View style={styles.shadowLight}>
    {/* 内容 */}
  </View>
</View>
```
shadowDark: {
  shadowColor: '#D1D9E6',
  shadowOffset: { width: 6, height: 6 },
  shadowOpacity: 0.7,
  shadowRadius: 8,
  borderRadius: 24,
  marginBottom: 16,
}
shadowLight: {
  shadowColor: '#FFFFFF',
  shadowOffset: { width: -6, height: -6 },
  shadowOpacity: 0.9,
  shadowRadius: 8,
  backgroundColor: '#F0F0F3',
  borderRadius: 24,
  padding: 20,
}
Android 降级：
  elevation: 6,
  backgroundColor: '#F0F0F3',
  borderRadius: 24,
  borderWidth: 0.5,
  borderColor: 'rgba(255,255,255,0.5)'

★ 凹陷效果（输入框/搜索栏/内嵌区域）★
{
  backgroundColor: '#E8E8EB',
  borderRadius: 16,
  padding: 16,
  borderWidth: 1,
  borderColor: 'rgba(255,255,255,0.6)',
}

★ 绝对禁止 ★
- ❌ 卡片用白色 #FFFFFF 背景
- ❌ 用 border 描边区分层级
- ❌ 选中态用矩形边框包裹
- ❌ 单层 View 做双阴影（iOS 只认一组 shadow）
- ❌ 渐变底色 opacity 超过 0.08
