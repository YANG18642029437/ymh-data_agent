【整体风格】
大地色系 + 有机圆润形状 + 手绘/植物纹理感

【适用应用】
环保/可持续、有机食品、花园管理、瑜伽冥想、户外徒步

【配色】
主色：#2D6A4F（森林绿）
辅色：#D4A276（陶土棕）
点缀色：#E9C46A（暖金黄）
背景：#FDF8F0（米黄暖白）
卡片：#FFFFFF
次级背景块：#F1EBE0（浅亚麻色）
文字主：#3D3229（暖棕黑）
文字次：#8B7D6B（灰棕）

【布局】
页面容器：flex: 1，backgroundColor: '#FDF8F0'
内容滚动：<ScrollView>
contentContainerStyle: paddingHorizontal: 20, paddingBottom: 100
栅格间距：marginBottom: 20
双列布局：flexDirection: 'row', flexWrap: 'wrap',
子项 width: '47%', marginHorizontal: '1.5%'
不规则大小混排：第一项 width: '100%' 高，后面双列矮，
形成杂志感瀑布
响应式：useWindowDimensions()

【核心样式手法】
有机圆角（非对称圆角）：
borderTopLeftRadius: 32,
borderTopRightRadius: 8,
borderBottomLeftRadius: 8,
borderBottomRightRadius: 32,
→ 产生"叶片/鹅卵石"不规则形状感

柔和阴影：
iOS: shadowColor: '#3D3229',
shadowOffset: {width: 0, height: 4},
shadowOpacity: 0.08, shadowRadius: 12
Android: elevation: 3

纹理底色块：
<View style={{
backgroundColor: '#F1EBE0',
borderRadius: 24,
padding: 20,
}}>
模拟纸质/亚麻感（可叠加 opacity: 0.06 的纹理图片）

线条插画风图标：
@expo/vector-icons 的 Feather 图标集
配合装饰圆圈:
<View style={{ width: 56, height: 56, borderRadius: 28,
backgroundColor: 'rgba(45,106,79,0.08)',
borderWidth: 1.5, borderColor: 'rgba(45,106,79,0.15)' }}>

手绘虚线：
borderWidth: 1.5, borderColor: '#D4A276',
borderStyle: 'dashed',（RN 支持 'dashed'）
borderRadius: 16

暖色渐变装饰：
<LinearGradient colors={['#2D6A4F','#52B788']}
start={{x:0,y:0}} end={{x:1,y:1}}>
用于按钮、头部区域

重叠圆形装饰：
两个 <View> 部分重叠，borderRadius: 9999,
分别 backgroundColor: 'rgba(45,106,79,0.1)' 和
'rgba(212,162,118,0.15)'，z轴层叠

【技术栈】
图标：@expo/vector-icons（Feather）
渐变：expo-linear-gradient
图片：expo-image（植物/自然图片需 WebP 优化）
字体：expo-font 加载手写/圆润字体: Quicksand、Comfortaa，
配合系统衬线字体做标题
动画：react-native-reanimated（轻微浮动呼吸动画、ScrollView 视差）
SVG：react-native-svg（波浪分割线、叶子形状装饰）
样式方案：StyleSheet.create
