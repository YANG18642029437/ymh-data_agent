【整体风格】
纯白大留白 + 极细线条 + 黑白灰为主 + 微妙强调色

【适用应用】
笔记/效率工具、阅读类、博客、极简电商、SaaS 管理

【配色】
主色：#111111（近纯黑）
强调色：#FF4444（极简红 — 仅用于极少量焦点）
背景：#FFFFFF
次级背景：#F7F7F7
卡片：#FFFFFF
边框：#ECECEC
文字主：#111111
文字副：#888888
禁用：#CCCCCC

【布局】
页面容器：flex: 1, backgroundColor: '#FFFFFF'
内容滚动：<ScrollView> contentContainerStyle: paddingBottom: 100
栅格间距：marginBottom: 24
正文区域：paddingHorizontal: 24
响应式：useWindowDimensions()，大屏加大留白

【核心样式手法】
极致留白（核心）：
组件间 marginBottom: 24~40
内容区 paddingVertical: 32
→ 空气感 = 高级感

极细分割线：
height: 1, 或 borderBottomWidth: 1,
borderColor: '#ECECEC'（或 StyleSheet.hairlineWidth）
→ 用线条而非阴影划分层级

无阴影，用边框区分：
卡片 borderWidth: 1, borderColor: '#ECECEC', borderRadius: 12
→ 扁平无阴影，极度克制

黑色实心按钮（唯一强调）：
backgroundColor: '#111', borderRadius: 12
→ 黑白对比即焦点

排版为王：
标题 fontSize: 28, fontWeight: '700', color: '#111', letterSpacing: -0.5
正文 fontSize: 15, fontWeight: '400', color: '#888', lineHeight: 24
辅助 fontSize: 12, color: '#CCCCCC'
→ 字号层级清晰，无装饰

【技术栈】
图标：@expo/vector-icons（Feather — 极细线性最佳选择）
渐变：一般不用（保持扁平）
字体：expo-font 加载 Inter 或 SF Pro 替代
动画：react-native-reanimated（
入场仅 opacity: 0→1, duration: 300,
Easing.out(Easing.quad)；极度克制，避免花哨）
样式方案：StyleSheet.create
